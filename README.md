**“群聊水群利器，让你的群友成为行走的表情包！”**

这是一个专为 [AstrBot](https://github.com/AstrBotDevs/AstrBot) 框架打造的趣味互动插件。通过本插件，你可以轻松抓取 QQ 群友的头像（或发送的图片），一键生成各种生草、可爱的静态与动态表情包。无论是想“迫害”群友，还是活跃群聊气氛，这都是你的不二之选！

## ✨ 核心特性

* **🖼️ 动静结合：** 支持生成 `GIF` 动态图和 `PNG` 静态图，满足不同场景的斗图需求。
* **🎭 海量模板内置：** 内置几十种高人气表情包模板，包含且不限于：
    * **动作类：** `吸`、`撅`、`敲`、`抛`、`揉搓`、`打拳`...
    * **二次元专属：** `可莉吃`、`草神啃`、`胡桃啃`、`泉此方看`、`白子舔`...
    * **互动类：** `贴贴`（支持双人头像！）、`抱抱`、`膜拜`、`抱大腿`...
* **🎛️ WebUI 完美兼容：** 所有的表情包指令均已接入 AstrBot 的可视化配置面板。管理员可以在后台随时开启或禁用特定的表情包功能，一切尽在掌握。
* **⚡ 极简触发：** 支持通过 `@目标群友` 或 `图片 + 指令` 的方式快速生成，无缝衔接日常聊天。

## 🚀 快速上手

### 触发方式
在群聊中，你可以通过以下两种方式呼叫机器人生成表情包：

1.  **艾特群友：** `/[指令] @某个群友`
    > 示例：`/吸 @群主` （机器人会自动抓取群主的头像并疯狂输出）
2.  **带图发送：** 发送一张图片，并配文 `/[指令]`
    > 示例：QQ聊天界面全屏输入一张图片并配文 `/舔屏`

### 热门指令一览
*(仅展示部分，完整开关请前往 AstrBot 管理面板查看)*

* `/敲` —— 咚咚咚！敲打群友的小脑瓜。
* `/贴贴` —— 飞扑上去，和目标群友的头像紧紧贴在一起。
* `/草神啃` —— 纳西妲也想尝尝这个头像的味道。
* `/科目三` —— 让群友的头像跟着音乐摇起来（社会摇准备）！
* `/卖掉了` —— 啪！成交！把群友无情卖掉。

## 📦 安装与配置

1.  在 AstrBot 的插件市场中搜索 `表情包制造厂` 并安装，或者将本仓库克隆至你的 `data/plugins/` 目录下。
2.  重启 AstrBot。
3.  进入 AstrBot 的 WebUI 管理面板，在插件配置中，你可以看到所有表情包的独立开关，按需开启/关闭即可。
